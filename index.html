Reversi

Tech Stack: Java, Java Swing, IntelliJ IDEA

I developed a Hexagonal Reversi game in Java, focusing on a modular, object-oriented design that’s scalable and well-tested. The game uses a clear MVC architecture for easy maintenance. It features a GUI that supports both mouse and keyboard inputs, creating a smooth, interactive experience for two players (human or AI). The AI player can be programmed with multiple strategies, adding variety and challenge to gameplay. In this demo, I am playing against an AI player.

Spelling Bee

Tech Stack: Java, Java Swing, IntelliJ IDEA

I recreated the New York Times Spelling Bee game, adding a customizable feature to let players choose which letters are used. I also built an anagram solver that finds all possible words based on the chosen letters and game constraints, making it easy to explore every solution within the puzzle’s rules.
